# Packages
 
 Ce dossier contient le code partagé réutilisable par plusieurs addons ou pipelines.
 
 Exemples : client Mealie, contrats de données, utilitaires de normalisation, composants communs.
 
 Package actuellement important :
 - `mealie-dev-stack/` : stack locale Docker canonique pour lancer Mealie en développement et tester les addons avant validation dans Coolify.
